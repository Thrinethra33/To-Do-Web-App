# To-Do Web Application

This repository contains the solution for **Task 04** of the SkillCraft Technology internship, as outlined in the assignment guidelines from IMG_8615.jpg. The project is a functional, intuitive web application designed to help users efficiently organize and manage their tasks.

## 🚀 Project Features

* **Task Management:** Seamlessly add, edit, and delete tasks to keep lists accurate and updated.
* **Organized Lists:** Structure and sort items efficiently to maintain clarity on upcoming goals.
* **Completion Status:** Mark tasks as completed with immediate visual feedback.
* **Deadlines & Schedules:** Set specific dates and times for individual tasks to stay on top of schedules.
* **Responsive Layout:** A clean, user-friendly interface optimized for cross-device responsiveness.

## 🛠️ Technologies Used

* **HTML5:** Provides a structured layout for text inputs, lists, buttons, and date/time pickers.
* **CSS3:** For styling individual task elements, responsive design, animations, and completion states (e.g., strike-through lines).
* **JavaScript (ES6):** Powers application CRUD logic, handles event listeners, dynamic DOM updates, and tracks task status arrays.
